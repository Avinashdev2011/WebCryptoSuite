# WebCryptoSuite - Encryption Analyzer & Security Tools

WebCryptoSuite - A comprehensive online suite for cryptographic functions including algorithm detection, hashing, symmetric encryption, and asymmetric key generation. All operations happen securely in your browser with no data sent to any server.

## Features

- **Algorithm Detection**: Identify hash types like MD5, SHA-1, SHA-256, SHA-512, Bcrypt, and RSA
- **Hash Generation**: Create SHA-256, SHA-512, and MD5 hashes
- **Password Security**: Bcrypt password hashing and verification
- **Symmetric Encryption**: AES encryption and decryption
- **Asymmetric Encryption**: RSA key generation, encryption, and decryption
- **Client-Side Only**: All processing happens in your browser for maximum security

## Technologies Used

- HTML5
- CSS3 (with Tailwind CSS)
- JavaScript
- CryptoJS library
- BcryptJS library
- JSEncrypt library

## Deployment

This application is deployed at https://webcryptosuite.vercel.app/ for fast, global access.

## Security

All cryptographic operations are performed client-side, ensuring your sensitive data never leaves your browser.
